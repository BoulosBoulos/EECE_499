# SUMO T-Intersection Scenario

## Layout

- **Ego:** Approaches from stem (south), turns right (east).
- **Other vehicle:** On bar (west to east), goes straight across.
- **Conflict:** Both use the connection from center to right_out.

```
       left (-80,0) =========> center (0,0) =========> right (80,0)
                                      ^
                                      |
                                    stem (0,-100)
                                      ^
                                    ego starts
```

## Files

Generated by `ScenarioGenerator` into `scenarios/sumo_t/`:

- `t.nod.xml` — nodes (plain XML)
- `t.edg.xml` — edges (plain XML)
- `t.net.xml` — compiled network (from netconvert)
- `t.rou.xml` — routes (ego_route, car_left_right, moto_right_left, etc.) and vTypes
- `t.sumocfg` — SUMO config

## Commands

```bash
# Generate scenario
python3 -c "from scenario.generator import ScenarioGenerator; ScenarioGenerator().generate('scenarios/sumo_t')"

# Visualize (headless)
make visualize-sumo

# Visualize with GUI (watch episodes)
make visualize-sumo-gui

# Train with SUMO
make train-sumo
```

## Prerequisites

```bash
sudo apt install sumo sumo-tools
export SUMO_HOME=/usr/share/sumo
```
