# Visualization and Analysis Infrastructure

## SUMO GUI Visualization

### Building Polygons
Buildings at the NW and NE corners are rendered as brown polygons in the SUMO GUI. Coordinates match the occlusion geometry in `sumo_env.py`:
- **NW building**: `(-3.5, 3.5) -> (-30, 3.5) -> (-30, 20) -> (-3.5, 20)`
- **NE building**: `(3.5, 3.5) -> (30, 3.5) -> (30, 20) -> (3.5, 20)`

Generated by `scenario/generator.py` as `t_buildings.poly.xml`.

With `--no_buildings`, building polygons are removed from the GUI at episode start.

### Visibility Overlay

```bash
python experiments/pde/visualize_sumo.py --gui --show_visibility \
    --checkpoint model.pt --method hjb_aux
```

Shows 20 colored dots in the conflict zone:
- **Green**: visible from ego position
- **Red**: occluded by building

Dots update every step as ego approaches the junction.

## Smoke Test

```bash
python experiments/pde/smoke_test.py --scenario 1a --ego_maneuver stem_right
```

Tests all 5 methods end-to-end:
1. Train for 512 steps
2. Verify checkpoint, CSV, and meta.json are created
3. Run 5 eval episodes
4. Report pass/fail per method

## Conflict Verification

```bash
python experiments/pde/verify_conflicts.py --episodes 50 --scenarios 1a 1b 2
```

Runs random-policy episodes and checks that agents come within 15m of ego in >= 70% of episodes. Tests conflict-guaranteed spawning across all maneuvers.

## Analysis Scripts

All analysis scripts are in `experiments/pde/analysis/`.

### Learning Curves

```bash
python experiments/pde/analysis/plot_learning_curves.py \
    --results_dir results/ablation/tier1 \
    --scenario 1a --maneuver stem_right \
    --out results/figures/learning_curves.png
```

Plots mean episode return vs training steps with shaded std bands, one line per method.

### PDE Convergence

```bash
python experiments/pde/analysis/plot_pde_convergence.py \
    --results_dir results/ablation/tier1 \
    --scenario 1a --maneuver stem_right
```

Plots PDE residual convergence and distillation gap for the 4 PDE methods.

### Results Tables (LaTeX)

```bash
python experiments/pde/analysis/generate_results_tables.py \
    --eval_dir results/ablation/tier1 \
    --out results/tables/
```

Generates LaTeX table with mean +/- std for return, collision rate, success rate, TTC. Includes Welch's t-test significance markers.

### Failure Trajectories

```bash
python experiments/pde/analysis/plot_failure_trajectories.py \
    --fail_dir results/pde/failures \
    --out results/figures/failures/
```

Plots ego trajectory (color-coded by action) and TTC timeline for collision episodes.

### Computational Overhead

```bash
python experiments/pde/analysis/compute_overhead.py \
    --results_dir results/ablation/tier1
```

Reports wall-time per training iteration for each method, relative to DRPPO baseline.

### Visibility Progression

```bash
python experiments/pde/plot_visibility_progression.py
```

Generates side-by-side figure of alpha_cz vs d_cz with and without buildings. Output: `results/figures/visibility_progression.png`.
